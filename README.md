# todays-camping (오늘의 캠핑) 이란?
### [오늘의 캠핑](https://todays-camping.vercel.app/)  
국내의 다양한 캠핑지를 인스타그램 피드 형태로 볼 수 있도록 만든 토이 프로젝트. 

# 기술 스택
### Front-end
- React
- Next.js
- Typescript

### Back-end
- Node.js
- Express
- postgresql

### etc
- git
- heroku
- vercel
- 공공데이터포털 Open API
- 카카오 Open API


# 배포 환경
- Frontend: Vercel
- Backend: Heroku
